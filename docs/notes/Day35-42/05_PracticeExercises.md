# Practice Exercises

DAY 35-42: Practice Exercises

Exercise 1: Debugging Drill
Pick a crash stack trace. Identify exception, first app frame, likely invalid assumption,
and what log/test would confirm it.

Exercise 2: Retry Policy
Implement retryWithBackoff that retries only transient errors and stops on auth/validation errors.

Exercise 3: Algorithm Practice
Solve in Kotlin:
- Two Sum
- Longest substring without repeating characters
- Valid palindrome
- BFS shortest path
- Climbing stairs

Exercise 4: Mock Interview
Time-box 60 minutes:
- 5 min intro
- 15 min Kotlin/lifecycle questions
- 15 min coroutines/architecture
- 15 min coding exercise
- 5 min your questions
- 5 min retrospective

Exercise 5: Project Story
Prepare two project stories using Context, Challenge, Decision, Trade-off, Result, Learning.
